How to get your API credentials:

  1. Log in to your Bybit account
  2. Go to Account & Security → API Management
  3. Create a new API key (select "Read" permission minimum)
  4. Copy the API Key and Secret
  5. Paste them above and click Save

Note: Your credentials are stored in a .env file on your local machine. Never share them with anyone!

Environment Options:
  • Demo: Use demo account credentials (api-demo.bybit.com)
  • Testnet: Use testnet for testing (api-testnet.bybit.com)
  • Neither: Production trading (api.bybit.com)
